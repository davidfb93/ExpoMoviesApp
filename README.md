## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Copy .env.template file, rename to .env and place the moviesDB api_key in the .env file.

   ```bash
    https://www.themoviedb.org/settings/api
   ```

3. Start the app

   ```bash
    npx expo start
   ```
